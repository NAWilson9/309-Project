boardGetHelper(req, res)

Helper function for get requests. If IDs are specified in the req body it queries the server to get the information for each board with a specified ID. If one is not found then an error is sent to the requesting client and execution stops. Otherwise all the boards information is sent back to the client. If a creator is specified it queries the server for all boards created by that user. If none are found it sends an error to the client. Otherwise it sends the found boards information to the client.

Name Type Description
req contains the information for the request
res the object to send results to

boardPostHelper(req, res)

Helper function for post requests. If no ID is specified in the req body it sends an error to the client. If an ID is specified it queries the server to see if a board with the given ID exists. If so the board is updated, if not then a board is created and stored. Then the resulting updated or new board is sent back to the client. If an error is encountered then the error is sent to the client and execution stops.
